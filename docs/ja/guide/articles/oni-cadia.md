# ONI-CADIA 紹介

[ONI-CADIA](https://github.com/Sunwood-ai-labs/ONI-CADIA) は、`onizuka-openclaw-autonomous-team-starter` を土台にした公開リポジトリです。

## この実例が役立つ理由

このスターターは、読むだけでなく派生プロジェクトへ展開することを想定しています。`ONI-CADIA` は、基盤となる runtime 構成を保ったまま、project 名、README 導線、docs surface を専用化して公開する時の形を示してくれます。

## このスターターから引き継いでいるもの

- `uv` と PowerShell を前提にした Windows-first の operator flow
- agent ごとに分離された OpenClaw pod と生成 state
- `SOUL.md`、`IDENTITY.md`、`USER.md`、`HEARTBEAT.md`、`TOOLS.md`、`BOOTSTRAP.md` などの workspace scaffold
- mention、smoke test、自律 chatter まで扱えるローカル Mattermost 会話ラボ

## 派生プロジェクト側で変えているもの

- 公開上の project identity を `ONI-CADIA` として見せていること
- quick start の作業パスを `D:\Prj\ONI-CADIA` に寄せていること
- repository、docs、release 向け導線をスターター名ではなく派生プロジェクト名でまとめていること

## どんな時の参考になるか

このスターターを fork して名前付きの自律チーム project に仕立てたい時、`ONI-CADIA` は runtime glue、persona scaffold、会話ラボの導線を大きく壊さずに引き継げることを示す実例になります。
